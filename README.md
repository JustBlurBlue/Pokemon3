 **这是一个多人宝可梦对站游戏** 

*警告：还并没有输入检测模块，所以一切输入都不能有差错* 

### ***游戏初始化*** 

**开始时可以选择真人玩家与PC的个数 **

需要用空格分隔 

例如

```
1 1
```

代表一个玩家与一个PC

**输入名字**

除了不可为"PC"之外

你可以取任意的名字

**选择宝可梦**

规则与选择   玩家一致

需要用空格分隔

Ep:

```
1 2 3
```

选择了妙蛙种子，皮卡丘，小火龙

****

### ***游戏过程***

每回合活着的玩家轮流操作

偶数回合为从大到小

奇数回合从小到大

**操作过程**

你需要按照打印出的玩家序号

先选择一个目标

然后再选择一个技能使用

### ***游戏结束***

当一个角色失去所有宝可梦时 会失败

当只剩下一个玩家时，系统会判定该玩家胜利
